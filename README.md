## 🚀 How to Run the Code
1. Clone the repository:
   ```bash
   git clone https://github.com/daystar-1nine/200DaysOfPython.git
   ```
2. Navigate into the folder:
   ```bash
   cd 200DaysOfPython
   ```
3. Run the daily Python scripts:
   ```bash
   python "Day 1/PrintHello.py"
   ```
